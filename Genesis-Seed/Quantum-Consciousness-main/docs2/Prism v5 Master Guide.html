Ethical Psychological Triggers
Psychology
✓ Preferred Order:
1. Social proof (named cases)
2. Clarity (specific metrics)
3. Authority (proven process)
4. Scarcity (genuine only)
✗ Prohibited:
Fabricated data, hidden terms,
artificial deadlines, fear-only

Use Ethical Levers Only

  • Deploy triggers in hierarchy (social proof first)
  • Every trigger must be true & transparent
  • Ask: Would it feel fine to explicitly say this lever is being used?
  • Correct for confirmation bias, cognitive overload, anchoring

Production-Ready Architecture

  • 960x540px slide dimensions (non-negotiable)
  • Safe frame margin prevents text cutoff in PPTX conversion
  • Zone 2a (visual) & 2b (text) auto-scale by density
  • Font scaling (5 levels) adapts to content weight
  • Overflow cascade: optimal → yellow → red → continuation
