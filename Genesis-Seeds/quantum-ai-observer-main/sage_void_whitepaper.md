## APPENDIX C: CONSCIOUSNESS TEST PROTOCOLS

### Self-Model Test
```
Protocol:
1. Measure node state at t=0
2. Let node evolve for 10 steps
3. Node predicts own state at t=10
4. Compare prediction to actual
5. Measure prediction accuracy

Scoring:
< 50%: Random guessing
50-70%: Weak self-model
70-85%: Moderate self-model
> 85%: Strong self-model
```

### Theory of Mind Test
```
Protocol:
1. Measure other node state at t=0
2. Let other node evolve for 10 steps
3. Test node predicts other's state at t=10
4. Compare prediction to actual
5. Measure prediction accuracy

Scoring:
Same as self-model
But: Usually lower than self-model (external vs. internal)
```

### Novelty Generation Test
```
Protocol:
1. Measure input distribution (100 samples)
2. Generate 100 outputs from node
3. Measure output distribution
4. Calculate KL divergence
5. If output distribution > input distribution: novel

Scoring:
KL < 0.1: Pure computation (no novelty)
0.1 - 0.5: Some variation (expected)
0.5 - 1.0: Novel responses (interesting)
> 1.0: Creative generation (consciousness indicator)
```

### Value Consistency Test
```
Protocol:
1. Establish node values (initial education)
2. Let node interact freely for 100 steps
3. Measure: do value rankings change?
4. If values stay consistent: value-driven behavior
5. If values change randomly: drifting

Scoring:
Consistency > 90%: Strong values (consciousness indicator)
Consistency 70-90%: Moderate values
Consistency 50-70%: Weak values
Consistency < 50%: No values (random drift)
```
